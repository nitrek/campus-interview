Java

What is the difference between JDK, JRE, and JVM?

JVM: An abstract machine that executes bytecode.

JRE: JVM + libraries needed to run Java programs.

JDK: JRE + tools for development (compiler, debugger, etc.).

Difference between String, StringBuilder, and StringBuffer?

String: Immutable, every modification creates a new object.

StringBuilder: Mutable, faster, not thread-safe.

StringBuffer: Mutable, thread-safe, slightly slower.

Node.js

What is Node.js?

Node.js is an open-source, cross-platform, back-end JavaScript runtime built on Google Chrome's V8 engine. It executes JS outside the browser.

What is NPM?

NPM is the Node Package Manager. It includes a CLI for managing packages and an online registry for open-source packages.

Explain the Node.js event loop.

The event loop allows Node.js to perform non-blocking I/O operations. Async operations are handled by the OS, and callbacks are queued to execute sequentially in the loop.

Difference between props and state?

Props: Data passed from parent, read-only.

State: Data managed within a component, mutable.

What is Virtual DOM?

Virtual DOM is a lightweight copy of the real DOM. React updates it first, then calculates minimal changes for the real DOM.

Python

Difference between list and tuple?

List: Mutable.

Tuple: Immutable, can be used as dictionary keys.

What are Python decorators?

Decorators wrap a function to add new functionality without changing the function's structure.
